## How are values compared? (i.e. comparing two string)
#### Java
| Comparison | Type |
| --- | --- |
| a == b, a != b | Reference |
| a.equals(b) | Value |
| a.compareTo(b) | Value |
| compare(a, b) | Value |
#### Swift
Comparison | Type
--- | ---
a == b, a != b | Value
a === b, a !== b | Reference
